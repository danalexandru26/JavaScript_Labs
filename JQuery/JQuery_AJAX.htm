JQuery AJAX

Introduction

AJAX stands for Asynchronous Javascript And XML. It allows us to load dynamic data to our webpages without actually reloading the whole page. As such, AJAX exchanges data with a backend server and can also be thought of as a series of methods that we can use in interacting with the deeper layers of our page.

AJAX is generally used together with JQuery. Using AJAX by its own can prove to be a difficult task. This is because different browsers requier different preambles for loading and properly executing AJAX functions. JQuery performs the dependency heavy lifting in our stead.

There are several AJAX methods of note, out of which load, get and post are the most important to us. We will go over each one in a separate section while providing examples to illustrate their usability.

JQuery Load

The JQuery load() method is a powerful AJAX tool that allows us to load data from a server and place it within a selected DOM element. It has the following basic structure:

function loadContent() {
    $("#result").load("ajax_content.html", function(response, status, xhr) {
        if (status == "success") {
            console.log("Content loaded successfully!");
        }
        else {
            console.log("Error: " + xhr.statusText);
        }
    });
}

The URL field is mandatory and it points to the file that we wish to load into an element (in this case identified by the result id).

The JQuery code that generated the bevahior above is the following: 

$(document).ready(function() {
    $("#AJAXLoad").click(function() {
        let display = false;
        $("#loadAJAXParagraph")
            .load("../auxiliary/ajaxLoadPara.txt");
        $(this).text(function(_, oldText) {
            if (oldText === "Load Paragraph") {
                display = true;
                return "Remove Paragraph";
            } else {
                display = false;
                return "Load Paragraph";
            }
        });
        $("#loadAJAXParagraph")
            .toggle(display);
    });
});
