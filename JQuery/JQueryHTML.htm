JQuery and HTML

JQuery is a powerful tool for HTML manipulation. We can use it to modify the content, structure and attributes of any HTML element of our choosing.

The term that defined the overall structure of an HTML document is the DOM (Document Object Model). The DOM represents the structure of an HTML document as a tree of objects, allowing JavaScript and jQuery to manipulate elements dynamically.

JQuery DOM Manipulation

There are several actions or events that we can use in order to manipulate the DOM. This list includes the following:

  • text(), useful for accessing the text content of selected objects.
  • html(), useful for accessing the content of html elements, including markup.
  • val(), useful for accessing the value of form elements.
  • append(), useful for adding elements at the end of the current selected tag.
  • remove(), useful for deleting elements and their children.

These are many more functions which we can call in order to set, add, and remove DOM elements.

Get Statements

These functions seem quite close in function. However, it is useful to remember that most of the content on a HTML page is text. As such, different functions provide different methods for text manipulation, in respect to the context of the tag that generated an event.

In order to make JQuery feel more natural, some examples of DOM manipulation will be provided in the code snippets below. 

$(document).ready(function() {
    console.log("jQuery has loaded!");
    $("#paraTest").click(function() {
        alert(
            "The paragraph has the following text: " +
            $("#paraTest").text()
        );
    });
});

Once the paragraph above is clicked upon, a new alert message, with the paragraph's text, will be generated. This JQuery is a textbook example of a get DOM statement.

Set statements

We can also write set statements in order to modify the content of desired tags. In order to achieve this, we use the same three functinos described above. The only substantial difference is that we have to define callback functions.

The callback function has two parameters of note. The first is the id of the current selected element in the list of available elements of our DOM structure. The second is the old value of the attribute, field or property that we wish to modify.

The generic structure of a JQuery set statement is described above.

$(document).ready(function() {
    $("#genericID").click(function() {
        $("#genericChild").text(function(id, oldText) {
            return oldText + " is deprecated";
        });
    });
});

As we can see, set statements are quite simple. What's even better is that the structure above is generally correct for most modifications we wish to do on our tags.

Add Statements

As has been described in the precedent paragraphs, we can use DOM statements to add content to our DOM elements. There is a variety of functions that we may use, depending on the position we wish our inserted element to have.

A list of add statements can be found below:

  • append(), useful for adding elements at the end of the current selected tag.
  • prepend(), useful for adding elements at the beginning of the current selected tag.
  • after(), useful for adding elements after the current DOM element.
  • before(), useful for adding elements before the current DOM element.
