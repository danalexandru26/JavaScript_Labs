JQuery and HTML

JQuery is a powerful tool for HTML manipulation. We can use it to modify the content, structure and attributes of any HTML element of our choosing.

The term that defined the overall structure of an HTML document is the DOM (Document Object Model). The DOM represents the structure of an HTML document as a tree of objects, allowing JavaScript and jQuery to manipulate elements dynamically.

JQuery DOM Manipulation

There are several actions or events that we can use in order to manipulate the DOM. This list includes the following:

  • text(), useful for accessing the text content of selected objects
  • html(), useful for accessing the content of html elements, including markup
  • val(), useful for accessing the value of form elements

These functions seem quite close in function. However, it is useful to remember that most of the content on a HTML page is text. As such, different functions provide different methods for text manipulation, in respect to the context of the tag that generated an event.
