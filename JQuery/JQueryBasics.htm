JQuery Basics

JQuery is a fast, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, and animation much simpler with an easy-to-use API that works across a multitude of browsers.

Getting Started with JQuery

To start using JQuery, you can include it in your HTML file by adding the following script tag within the <head> section:

$(document).ready(function() {
    $("p").click(function() {
        $(this).hide();
    });
});

This is only a sample and basic JQuery example. All it does is hide the clicked paragraph. As we can see, its format is human-readable and easy to understand. JQuery offers a beautiful pattern for structured understanding.

For a more thorough list of features, please access the JQuery documentation, by clicking on the following hyperlink: JQuery Function Selection.

Adding JQuery to our Project

When it comes to adding JQuery to our project, we have two options:

  • Using a CDN (Content Delivery Network)
  • Downloading and Hosting Locally

Using a CDN is simpler, all we have to do is add the following line to our HTML file:

<head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>
</head>

Alternatively, we can download the jQuery library and host it locally on our server. All we need to do is link its file location.

<head>
    <script src="jquery-3.7.1.min.js"></script>
</head>

JQuery Syntax

We write JQuery in order to manipulate or perform certain activities once an element is accessed in certain ways by the user. The formal syntax structure is the following:


$(selector).action()

These elements have the following meaning:

  • $: This symbol is used to access jQuery.
  • selector: This is used to "find" (or select) HTML elements.
  • action(): This is the jQuery action to be performed on the selected element(s).

The selector element is used to query for a specific element (defined by a tag name, id, class, or other attributes).

The action() element is used to perform a specific action on the selected element(s).

In the first JQuery example on this page, the ready action is used to specify that we want to execute the code within only once the page has finished loading. This prevents accidental execution and certain unexpected behaviors.

JQuery Samples

JQuery selectors can be any of the following element-defining attributes:

  • Element Selector: Selects elements based on the element name. Example: $("p")
  • ID Selector: Selects a single element based on its ID. Example: $("#myId")
  • Class Selector: Selects elements based on their class attribute. Example: $(".myClass")
  • Attribute Selector: Selects elements based on a specific attribute. Example: $("[type='text']")

Do keep in mind that the elements need not be hierarchically related. Within the body of a JQuery function, we can access any other html selector. Its presence only signifies that this element is changed once a certain action is triggered.

For example, this paragraph will be toggled once the button below is clicked.

The JQuery code for the function that defined the behavior is as follows:

$(document).ready(function() {
    $("#exampleButton").click(function() {
        $("#JQpara").toggle();
        $(this).text(function(_, currentText) {
            return currentText === "Hide Paragraph" ? "Show Paragraph" : "Hide Paragraph";
        });
    });
});
