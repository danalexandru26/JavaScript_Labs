JQuery Traversal

Introduction

We can use JQuery statements to iterate over the elements of our DOM's tree structure. This means we can access the children, parents or siblings of a selected element.

A basic sample structure of the ordinary DOM can be found below. Note the hierarchy of objects and their subordination.

The functions that we may use for accessing parents are as follows:

  • parent() - returns the direct hierarchical parent of the current selected tag.
  • parents() - returns the full list of parents, up to the root <html> tag.
  • parentUntil()- returns the list of parents between the selected element and the one provided as a function argument.

On the other hand, the functions we use for accessing children nodes are the following:

  • children() - returns the list of children the selected element has.
  • find() - returns a list of all the children that respect a selector parameter. This parameter can be any of the characteristics of a tag/

When it comes to accessing sibling elements, the choice of functions is more varied. For a full list of these methods please access the following hyperlink: Sibling Access.

The most useful methods, for which samples will also be provided are the following:

  • siblings() - this method will return a list of all the siblings our selected element has. Optional selectors can also be provided as function parameters .
  • next() - this method will return the next sibling (if it exists), in the order it appear in the DOM. .
  • prev() - this method will return the previous sibling (if it exists), in the order it appear in the DOM. .

A broader definition of all methods and their intent of use is found at the next hyperlink: JQuery Traversal.

Parent Traversal

Examples of parent traversal, in the form of code snippets, can be found below.

function parentTraversal() {
    $(document).ready(function () {
        let parent = $("#childParagraph").parent();
        let parentsList = $("#childParagraph").parents();
        let parentsUntil = $("#childParagraph").parentsUntil("div");

        console.log(parent);

        for (let i = 0; i < parentsList.length; i++) {
            console.log(parentsList[i]);
        }

        for (let i = 0; i < parentsUntil.length; i++) {
            console.log(parentsUntil[i]);
        }
    });
}

Child Traversal

Iterating over children is just as simple. Please consult with the examples below for a better understanding.

function childTraversal() {
    $(document).ready(function () {
        let childrenList = $("#introduction").children();
        let checkChild = $("#introduction").find("p");

        console.log("Children: " + childrenList);

        if (checkChild.length) {
            console.log("Found the child(ren)");
        }
        else {
            console.log("No such children have been found");
        }
    });
}
