Javascript Syntax and Variables

For all those who are familiar to how programming languages work and are written, Javascript is not much different, following a nearly identical structure to dynamically typed languages such as Python. Do not get fooled by its tricky name, it is rather most unlike Java, machine code discounted.

Javascript Syntax

The examples presented in the previous section of the tutorial have already offered us a nice insight into how the syntax is written in Javascript. Statements are separated by semicolons and are generally written on a line each, so our code may be cohesive and readable. In order to make everything clear and offer as much diversity as possible, some supplementary examples are provided below:

let a = 10;
let b = 20;

var c = 10;
var d = 20;

let sum = a + b;
var product = c * d;

console.log("Sum: " + sum);
console.log("Product: " + product);

let myName = "Explorer";
    
if (myName === "Explorer") {
    var greeting = "Welcome, Explorer!";
    console.log(greeting);
}
