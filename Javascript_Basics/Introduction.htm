Why Javascript?

Javascript, since its inception, has become the scripting language of the world wide web. Without going too much into the deep end on why this is so, consider the following idea: if php is the langauge we use when dealing with server-side processing of data, html the language we use for structuring the outline of our web-page and css the language that is used for styling the contents of our html tags, then Javascript shall be the language we use for modeling our web-page's behavior.

In other words, Javascript is what we use for dynamically manipulating the contents of our web-page. In order to achieve this, we may call upon the many methods that Javascript provides.

Throughout this whole tutorial series, we will explore these methods and how they can be used to create dynamic and interactive web-pages.

Integrating Javascript

There are multiple methods of integrating Javascript into an HTML document:

  • Inline: Using the script tag directly within the HTML element.
  • Internal: Using the script tag within the head or body section of the HTML document.
  • External: Linking to an external JavaScript file using the src attribute in the script tag.

Conventionally, the fastest way of loading a web-page that has Javascript scripts embedded is by using a linked external file. If we still desire to use it within our document, it's suggested to place the <script> tag in the head section of the HTML document. In case the <script> tag is used within the body section, it's recommended to place it at the end of the body for optimal performance. This is because the document is loaded top to bottom.

Inline JS

A text example of inline javascript can be found in the code block below.

<script>
function shipCategoryChange() {
    var shipCategory =
        document.getElementById("shipCategory").value;

    if (shipCategory == null) return;

    else if (shipCategory == "civilian") {
        var shipType = [
            'Fuel Tankers',
            'Civilian Transport Vessels',
            'Cargo Freighters'
        ];
    }
    else if (shipCategory == "military") {
        var shipType = [
            'Fighters',
            'Corvettes',
            'Frigates',
            'Destroyers',
            'Cruisers',
            'Carriers'
        ];
    }
    else return;

    var select =
        document.getElementById("shipType");
    select.innerHTML = null;

    shipType.forEach(function (type) {
        var option =
            document.createElement('option');
        option.value =
            type.toLowerCase().replace(/ /g, '_');
        option.innerHTML = type;
        select.appendChild(option);
    });
}
</script>

Outputting Javascript

There are multiple methods of outputting data using Javascript. The most common methods are:

  • console.log()
  • alert()
  • document.write()
  • document.getElementById().innerHTML = "..."

A brief explanation of what each of these method does can be foud below.

  • console.log() outputs data to the browser console.
  • alert() displays an alert dialog box with a message.
  • document.write() writes text directly into the HTML document.
  • document.getElementById().innerHTML = "..." changes the content of an element with a specific ID.
function logData() {
    var userName =
        document.getElementById("username").value;
    var userPassword =
        document.getElementById("password").value;
    var protected =
        document.getElementById("protected").checked;

    if (protected == false) {
        console.log(
            "Username: " + userName +
            " Password: " + userPassword
        );
    }
    else {
        console.log("The user is protected.");
    }
}

function setInnerPreformat() {
    var preformatTag =
