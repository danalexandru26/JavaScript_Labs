Conditionals and Loops represent some of the most important means of data manipulation in any programming language. Conditionals allow the program to make decisions, while loops enable repetitive actions. Without such structures, we would be writing repetitive code that's monstrously inefficient. Algorithms, minus those that are the least complex, being virtually impossible to write.

This document will provide a short compendium on the basic concepts of conditionals and loops in JavaScript by offering samples of each.

Conditionals

Conditionals are statements that perform different actions based on whether a specified condition evaluates to true or false. In JavaScript, the primary conditional statements are if, else if, and else. Here's a basic example: 

function sampleConditionals() {
    let nameA = "Bella";
    let nameB = "Charlie";

    if (nameA === nameB) {
        console.log(
            "The content of these two variables is identical."
        );
    }

    if (nameA == null) {
        console.log("nameA is null.");
    }

    if (nameB == null) {
        console.log("nameB is null.");
    }

    let score = 85;

    if (score >= 90) {
        console.log("Grade: A");
    }
    else if (score >= 80) {
        console.log("Grade: B");
    }
    else if (score >= 70) {
        console.log("Grade: C");
    }
    else if (score >= 60) {
        console.log("Grade: D");
    }
    else {
        console.log("Try again next year!");
    }
}

JavaScript also implements a ternary operator for concise conditional expressions. It has the structure and functions akin to its C++ analog. Please check the example below.

    let age = 20;
    let canVote = age >= 18 ? "Yes" : "No";
    console.log("Can vote: ", canVote);

In order to avoid writing pesky and slow if-else chains, the use of switch statements is suggested. Here's an example of a switch statement in JavaScript: 

switch (score) {
    case 90:
        console.log("Excellent work!");
        break;

    case 80:
        console.log("Good job!");
        break;

    case 70:
        console.log("You passed.");
        break;

    case 60:
        console.log("Barely made it.");
        break;

    default:
        console.log("Failed.");
        break;
}

Loops

Loops help us avoid writing lengthy repetitive code by allowing us to execute a block of code multiple times based on a condition. JavaScript provides several types of loops, including for, while, and do...while. Here are examples of each: 

function loops() {
    let nameArray = [
        "Bella",
        "Charlie",
        "Daisy",
        "Max"
    ];

    for (let i = 0;
         i < nameArray.length;
         i++) {

        console.log(
            "Name " +
            (i + 1) +
            ": " +
            nameArray[i]
        );
    }

    let sum = 0;
    let numsArray = [
        5,
        10,
        15,
        20,
        25
    ];
    let index = 0;

    while (index < numsArray.length) {
        sum += numsArray[index];
        index++;
    }

    console.log(
        "Total Sum: " + sum
    );

    let count = 0;
    let position = 0;
    let positionsArray = [
        2,
        4,
        6,
        8,
        10
    ];

    if (positionsArray.length > 0) {
        do {
            count += positionsArray[position];
            position++;
        } while (position < positionsArray.length);
    }

    console.log(
        "Counted Sum: " + count
    );
}

Because the condition in a do...while loop is checked after the loop body executes, it ensures that the loop body runs at least once, even if the condition is false initially. Moreover, if using do...while to iterate over an array(which is not generally recommended), it may be useful to nest the loop inside a conditional statement. In our case, it's nested inside an if statement.
